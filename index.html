An Introduction to Recommendation Systems in Python¶

Skill level: Intermediate

Tutorial requirements¶

  • Python 3.6+
  • Jupyter Lab
  • numpy
  • scikit-learn

Alternatively, you can also use Google’s new colab platform which allows you to run a Jupyter notebook environment in the cloud. You won’t need to locally install any of the above; however, you will need a gmail account.

Description¶

In this tutorial, we will explore the different types of recommendation systems and their implementations. We will also build our own recommendation system using data from the MovieLens database.
